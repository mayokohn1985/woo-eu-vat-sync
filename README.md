# Woo EU VAT Sync

Simple plugin to update EU VAT rates in WooCommerce.

## Features
- Manual VAT update
- Basic EU VAT rates
- No bloat

## Warning
This plugin updates standard VAT rates only.
Always verify tax compliance for your business.
